# Binary-Data-Processor-Python-C
Проект реализует межъязыковое взаимодействие через бинарный файл. Python создает данные, C-программа эффективно сортирует их (in-place), используя fseek/fwrite. Финальный Python-скрипт читает результат, распаковывает байты, используя `struct`, и выводит отсортированные записи о студентах в форматированный текстовый файл.
